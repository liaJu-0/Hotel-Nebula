# Hotel-Nebula

## Sobre o Projeto

O Hotel Nebula é um projeto de Banco de Dados desenvolvido para modelar e implementar um sistema de gestão hoteleira.

O objetivo é organizar e centralizar informações relacionadas à operação de um hotel, permitindo o controle de hóspedes, quartos, reservas, hospedagens, pagamentos, funcionários, serviços e avaliações dos clientes.

Este projeto percorre todas as etapas fundamentais de desenvolvimento de um banco de dados relacional, desde a modelagem até a criação de consultas para extração de informações gerenciais.

## Funcionalidades

- Cadastro de hóspedes
- Gerenciamento de quartos
- Controle de reservas
- Registro de hospedagens
- Controle de pagamentos
- Cadastro de funcionários
- Registro de serviços extras consumidos
- Avaliações e comentários dos hóspedes
- Consultas para apoio à gestão do hotel

## Estrutura do Projeto

O projeto está dividido em três etapas:

### Parte 1 - Modelagem de Dados
- Levantamento das entidades
- Identificação dos relacionamentos
- Criação do modelo conceitual (ER)
- Criação do modelo lógico relacional

### Parte 2 - Implementação do Banco
- Criação do banco de dados
- Criação das tabelas
- Definição de chaves primárias e estrangeiras
- Inserção de dados para testes

### Parte 3 - Consultas SQL
- Disponibilidade de quartos
- Histórico de reservas
- Faturamento mensal
- Serviços mais utilizados
- Avaliações dos quartos
- Reservas canceladas
- Relatórios de hospedagens por funcionário

## Tecnologias Utilizadas

- MySQL
- MySQL Workbench
- DBDiagram
- SQL

## Objetivo

Aplicar conceitos de modelagem de dados, bancos de dados relacionais e linguagem SQL por meio da construção de um sistema de gestão hoteleira completo.
